# Modern Landing Page

A beautiful, responsive landing page built with HTML, CSS, JavaScript, and Bootstrap 5.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Modern UI**: Clean and professional design with smooth animations
- **Bootstrap 5**: Leverages the latest Bootstrap framework for rapid development
- **Interactive Elements**: Smooth scrolling, hover effects, and form validation
- **Key Sections**:
  - Navigation bar with smooth scrolling links
  - Hero section with call-to-action buttons
  - Features showcase with 6 feature cards
  - Services section with detailed service offerings
  - Contact form with validation
  - Footer with social media links and contact information
  - Back-to-top button

## Technologies Used

- HTML5
- CSS3 (Custom styling with gradients and animations)
- JavaScript (ES6+)
- Bootstrap 5.3.2
- Bootstrap Icons

## Getting Started

### View the Landing Page

Simply open `index.html` in your web browser, or serve it using a local web server:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Node.js
npx serve

# Using PHP
php -S localhost:8080
```

Then navigate to `http://localhost:8080` in your browser.

## Customization

### Colors
Edit the CSS variables in `styles.css` to change the color scheme:

```css
:root {
    --primary-color: #0d6efd;
    --secondary-color: #6c757d;
    --dark-color: #212529;
    --light-color: #f8f9fa;
}
```

### Content
Update the text content directly in `index.html` to match your needs.

### Styling
Modify `styles.css` to adjust fonts, spacing, animations, and more.
